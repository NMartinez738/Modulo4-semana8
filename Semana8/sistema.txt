El comando "uname -a" muestra información sobre el sistema operativo, el kernel, la arquitectura del equipo y otros datos técnicos.

El comando "whoami" muestra el nombre del usuario que tiene iniciada la sesión en el sistema.

Conocer esta información es útil para administrar un servidor porque permite identificar el entorno donde se trabaja, 
verificar el usuarioque ejecuta los comandos y facilitar tareas de mantenimiento, configuración y solución de problemas.
